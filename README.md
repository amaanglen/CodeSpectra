# CodeSpectra

CodeSpectra is a simple and user-friendly tool for comparing two code files. It lets you upload files, view the differences line-by-line, and edit the code directly within the website. The goal is to make debugging and reviewing code faster and easier.

---

## Features
- Clean and minimal interface  
- Highlights differences between two code files  
- Allows users to create accounts and save their projects  
- Supports uploading folders and updating project files  

---

## Tech Stack
- React  
- Node.js  
- Express  
- MongoDB  
- Firebase  

---

This project was built to provide a quick and efficient way for developers to compare and edit code in one place.
